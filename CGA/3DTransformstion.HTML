3D TRANSFORM — ROTATION

Rotation changes the orientation of an object around the X, Y, or Z axis. Implemented as 4×4 matrices using cosine and sine functions — the mathematical backbone of every 3D graphics pipeline.

Core Concept · Transformation 02

3D TRANSFORM — TRANSLATION

Translation moves an object from one position to another in 3D space. It is represented by adding a displacement vector (tx, ty, tz) to the object's current coordinates.

Unlike rotation, translation does not change the orientation or size of an object — it simply shifts it. In matrix form, translation is applied using a 4×4 homogeneous transformation matrix.

X axis Y axis Z axis
T(tx,ty,tz) = [ 1 0 0 tx ] [ 0 1 0 ty ] [ 0 0 1 tz ] [ 0 0 0 1 ]

3D TRANSFORM — SCALING

Scaling changes the size of an object by multiplying its coordinates by scale factors (sx, sy, sz). Uniform scaling applies the same factor to all axes; non-uniform stretches them independently.

Drag the sliders to apply non-uniform scaling along each axis. Watch how the torus deforms in real-time — demonstrating the diagonal nature of the scaling matrix.

Mechanics

HOW IT WORKS

01

Matrix Composition

Every transform is a 4×4 matrix. Chaining transforms multiplies matrices left-to-right, enabling complex combined movements in a single draw call.

02

WebGL Pipeline

Three.js abstracts the WebGL vertex shader. Each vertex position is multiplied by the Model-View-Projection matrix before rasterisation.

03

User Input Mapping

Mouse drag events map to Euler angle deltas. Keyboard and slider inputs map to translation vectors and scale factors in real time.

04

requestAnimationFrame

Each animation frame calls renderer.render(), updating the scene graph at the display's refresh rate — targeting 60fps for fluid motion.

07
Real World

APPLICATIONS OF 3D TRANSFORM

🎮

Video Games

Real-time transforms power every object in engines like Unity and Unreal — from character movement to camera rigs.

🥽

Virtual Reality

Head tracking is pure rotation matrix math — mapping sensor quaternions to the camera transform 90× per second.

🎬

Animation & Film

CGI character rigs use hierarchical transform trees — each bone is a local transform matrix relative to its parent.

🛸

Simulation

Flight simulators encode aircraft attitude as rotation matrices, enabling physically accurate 6-DOF motion.

🌐

Interactive Web

Browser-based 3D experiences — from product visualisers to data maps — all rely on the same WebGL matrix math.

🔬

Scientific Vis

Molecular viewers, CT scan renderers, and physics simulations all transform 3D data into meaningful 2D projections.
